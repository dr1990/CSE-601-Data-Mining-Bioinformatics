
Script names corresponding to each algorithm:
PCA   : pca_script.py
SVD   : svd_script.py
s-sne : tsne_script.py

* Input the data file name to the respective scripts at the following placeholders:
Ex:
PCA: file = "pca_c.txt"
SVD: file = "pca_c.txt"
t-SNE: file = "pca_c.txt"

* Run the following commands to run each algorithm:
python pca_script.py
python svd_script.py
python tsne_script.py

* Scatter Plots are generated after each run.
